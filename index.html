About Linux

Linux is an open-source operating system.

  • Free and very easy to use
  • Command line based operating system
  • Very popular in the world of servers and embed programming

Linux began in 1991 as a personal project by Finnish student Linus Torvalds: to create a new free operating system kernel. The resulting Linux kernel has been marked by constant growth throughout its history. Since the initial release of its source code in 1991.

Why Linux? Why Linux and nothing else?

  • 01 Project open-sorce

    Everyone can contribute and extend this operating system.The kernel, all code of this system can be seen on the github site of the creator Linus Torvalds.
    See here

  • 02 Linux is everywhere

    Yes, Linux is everywhere. From the smallest device to the largest supercomputer, Linux is everywhere. It can be a car, router, phone, medical devices, plane, TV, satellite, watch or school tablet, Linux will be there.

  • 03 Freedome and Freedome

    Linux can be installed and used it as a desktop, firewall, a file server, or a web server. Linux allows a user to control every aspect of the operating systems. As Linux is an open-source operating system, it allows a user to modify its source (even source code of applications) itself as per the user requirements.
